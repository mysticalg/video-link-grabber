# Video Link Grabber (Chrome Extension, MV3)

Scans the current tab for likely video file links, stream manifests, and poster images, then lets you open or copy the links it finds.

Important: this extension does not bypass DRM, paywalls, login requirements, or protected streams. Only access media you own or have permission to use, and respect each site's terms.

## Features
- Scans only the active tab when the user opens the popup.
- Detects likely video sources from `<video>` and `<source>` tags, metadata, page links, and raw HTML.
- Shows poster or Open Graph thumbnails when they are available.
- Labels direct files, stream manifests, blob URLs, and data URLs clearly.
- Keeps permissions minimal: `activeTab` plus `scripting`.

## Install (Unpacked)
1. Download and unzip this folder.
2. In Chrome, open `chrome://extensions`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the `video-link-grabber` folder.
5. Pin the extension if you want quicker access.

## Notes
- Some pages expose only manifest URLs such as `.m3u8` or `.mpd`; those are playlists rather than single video files.
- `blob:` links are generated by the current page and may stop working after the tab reloads.
- Cross-origin thumbnails may fail to load; a placeholder is shown instead.
